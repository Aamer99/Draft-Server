# Draft-Server
Clone the project

```bash
  git clone https://link-to-project](https://github.com/Aamer99/Draft-Server.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
# Database Configuration
